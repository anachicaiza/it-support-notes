# Internet Troubleshooting Guide

Objective:
Troubleshoot internet connectivity issues systematically.

Commands Used:

- ipconfig: checks IP configuration.

- ping: tests connectivity.

- nslookup: tests DNS resolution.

---

# Troubleshooting Steps

1. Verify physical connection
2. Check IP address
3. Ping gateway
4. Ping external address
5. Test DNS resolution
6. Restart adapter if necessary
7. Escalate if unresolved
