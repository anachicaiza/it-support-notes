# Active Directory Basics

## What Is Active Directory?

It's a Microsoft directory service used to centrally manage users, computers, permissions and network resources within an organization.

It allows IT administrators to control authentication, user access and system management across a company network.



# Important Active Directory Concepts

## Domain

A domain is a centralized network environment managed through Active Directory.

Example:
A company may have all employee computers connected to one corporate domain.



## User Account

A user account allows a person to authenticate and access company systems and resources.

Common IT Support Tasks:

* Password resets
* Unlocking accounts
* Disabling accounts



## Groups

Groups are collections of users that share permissions and access rights.

Example:

* HR Group
* IT Support Group
* Finance Group

Using groups makes permission management easier.



## Organizational Units (OUs)

Organizational Units are containers used to organize users, computers and groups inside Active Directory.

They help administrators manage departments or locations separately.

Example:

* London Office OU
* HR OU
* IT Department OU



# Why Active Directory Is Important

Active Directory helps organizations:

* centrally manage users
* control access permissions
* improve security
* simplify administration

It is one of the most commonly used technologies in IT support environments.
