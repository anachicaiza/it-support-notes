# Task Manager

- Purpose: used to monitor system performance, running applications and background processes.

- Common Uses: 
* End frozen applications
* Check CPU and RAM usage
* Identify resource-heavy processes
* Monitor startup applications


# Device Manager

- Purpose: used to manage hardware devices and drivers installed on the system.

- Common Uses: 
* Identify driver issues
* Update drivers
* Disable problematic devices
* Troubleshoot hardware problems

# Important Note: Yellow warning symbols may indicate driver or hardware issues. #


# Services

- Purpose: manages background Windows services required for operating system functions.

- Common Uses:
* Restart services
* Troubleshoot printing or networking issues
* Manage automatic services

- Example Services:
* Print Spooler
* DHCP Client
* Windows Update


# Event Viewer

- Purpose: used to review Windows system and application logs.

- Common Uses:
* Investigate crashes
* Analyze system warnings and errors
* Troubleshoot operating system issues
